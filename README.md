# InvMan

### Installation
##### Create a virtual environment
`python3 -m venv env`
##### Activate the environment
`. env/bin/activate`
##### Install the project in the virtual environment
`pip install -e .`
##### Run the project
`export FLASK_APP=invman`
###### Initialize the database and run
`flask init-db`
`flask run`
